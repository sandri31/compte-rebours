## Description

Voici un projet d'une application de compte à rebours.  
Il y a CRUD pour mettre en place un ou plusieurs compte à rebours avec un timer qui démarre automatiquement quand on en crée un.  
Et un message qui s'affiche quand le timer est terminé.  

Cette application utilise Ruby on Rails 7, Stimulus.
